# ISC-S1

# GLOSARIO
## 1.	SISTEMA DE COMPUTO: conjunto de elementos que trabajan juntos para procesar información.
## 2.	HARDWARE: Componentes físicos de la computadora que se pueden ver y tocar, como teclado, monitor o CPU
## 3.	SOFTWARE: Programas que permiten que el hardware funcione y que el usuario realice tareas específicas.
## 4.	DATO: Valor o elemento aislado sin contexto que necesita ser procesado para tener significado.
## 5.	INOFRMACION: Conjunto de datos organizados y procesados que tienen significado y utilidad.
## 6.	ENTRADA: Información o datos que se introducen en un sistema para ser procesados.
## 7.	PORCESO: Conjunto de acciones que transforman los datos de entrada en información útil.
## 8.	SALIDA: Resultado que el sistema genera después de procesar los datos.
## 9.	ALMACENAMIENTO: Lugar donde se guardan datos e información para su uso posterior.
## 10.	BASE DE DATOS: Sistema organizado para almacenar, gestionar y consultar grandes cantidades de información.
## 11.	SISTEMA OPERATIVO: Programa principal que administra el hardware y permite ejecutar otros programas.
## 12.	COMPUTACION EN LA NUBE: Uso de servicios tecnológicos a través de internet para almacenar datos o ejecutar aplicaciones.
## 13.	INTELIGENCIA ARTIFICIAL: Tecnología que permite a las máquinas simular procesos de aprendizaje y toma de decisiones humanas.
## 14.	CIBERCIGURIDAD: Conjunto de prácticas y tecnologías que protegen sistemas y datos contra accesos no autorizados o ataques.
## 15.	RED DE COMPUTADORAS: Conjunto de dispositivos conectados entre sí para compartir información y recursos.
